# Go Cron Scheduler

This is a program to manage and run cron jobs, a cron jobs trigger an action like an api call.

Project that I have yet to start. 
Written in Go to learn a little bit Go.


## Objectives
#### Managing the jobs
- Add a cron job using an api call and store that somewere
- Delete the cron job

#### Running the jobs
- Run the cron job using goroutines
- No more than one job should run (use lock)

#### Infrastructure
- Should run this on docker
  - Via environment variable set the job
- Run tests on gh actions

#### Other
- Unit and E2E testing
- Written in Go
- No AI should be used
- Have fun
